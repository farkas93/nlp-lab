# Legacy Post-Training Area

This folder contains legacy and exploratory post-training scripts that are not part of the production manifest-driven SFT path.

Contents include older DPO/GRPO/ORPO runners, tutorial scripts, and helper formatters that were previously located under `src/`.

Use this area as reference code or for ad hoc experiments. The supported production SFT workflow is implemented under `src/eliza_trainer/sft/` and documented in `docs/runs/sft.md`.
